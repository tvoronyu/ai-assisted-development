# AI-Assisted Development — Homework

Репозиторій з домашніми завданнями курсу **GenAI and Agentic AI for Software Engineering**.

Кожне ДЗ знаходиться в окремій директорії `homework-{N}/` та подається через Pull Request у цей же репозиторій (base: `main`, compare: `homework-{N}-submission`).

## Структура

```
homework-1/    # ДЗ 1
homework-2/    # ДЗ 2
...
```

Кожна директорія містить `TASKS.md` з умовою, `README.md`, `HOWTORUN.md`, `src/`, `docs/screenshots/`, `demo/`.

## Як запустити конкретне ДЗ

Див. `HOWTORUN.md` у відповідній директорії `homework-{N}/`.

## Процес подачі

1. Робота ведеться на гілці `homework-{N}-submission`.
2. Коли готово — Pull Request у `main` цього репо.
3. Reviewer: `Alexey-Popov`.
